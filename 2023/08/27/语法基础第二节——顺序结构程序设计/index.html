语言基础（二）——顺序结构程序设计

我们已经学会了如何编写代码让计算机输出内容和计算简单的算术表达式。这一节我们将学习通过引入变量数据类型的概念，来介绍顺序结构程序设计。

所谓顺序结构程序设计，就是在编写程序时，将任务拆分成一条条的语句，按照顺序一步一步地执行这些语句。

一.变量的数据类型

小A中午准备做烤全鸡，为了吃上香喷喷的烤全鸡，小A准备了一只整鸡，一瓶烤肉酱，一碟佐料。小A把鸡放在了大碗里，烤肉酱放在了瓶子里，佐料放在了小碟子里。

在这个事例中，鸡、烤肉酱、佐料我们可以将其看作数据，而大碗、瓶子、小碟子我们可以看作变量。鸡、烤肉酱、佐料所需的容器不同，小A不会把鸡放在碟子里，也不会把佐料放在瓶子里。在C++中同样如此，变量有许多类型，不同的变量存储的数据精度和范围不同，在实际应用中我们应根据数据的实际情况合理选择变量类型。（不然一碟烤全鸡一定很滑稽○( ＾皿＾)っ）

首先我们先来了解了解常见的数据类型：

数据类型 占用空间 取值范围
int 4字节，32位 -2的31次方~2的31次方-1
float 4字节，32位 大约指数绝对值不超过37
double 8字节，64位 大约指数绝对值不超过307
char 1字节，8位 -128~127
long long 8字节，64位 -2的63次方~2的63次方-1
unsigned long long 8字节，64位 0~2的24次方-1
unsigned int 4字节，32位 0~2的32次方-1

表中取值范围中的“大约”是保守的估计，这个范围不是死的，有时可能能超过一点，有时又不能，但是不超过这个范围是安全的。另外可以看出，这些数据类型都是带符号的（signed），有正有负有0，加了unsigned的表示无符号数，只能表示0和正数，不过数据范围比带符号数大了一倍。

现在我们来挨个认识一下这些类型：

int:在上一节已经使用过了，只是没有提及，int类型用来存放整数。同样用来存放整数的还有long long ，后者能够表示的数据范围更大。

问：那能不能只用这些大的数据类型而不用小的数据类型呢？

答：不行，计算机内存有限，大的数据类型占用空间更大。

float：float用来存放浮点数（还记不记得这个名词），同样用来存放浮点数的还有double，后者能表示的数据范围更大。

char：char用来存放字符。
